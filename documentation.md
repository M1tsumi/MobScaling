# Mob Scaling Datapack Documentation

## Core Features
- Vanilla compatible mob scaling system
- 9 configurable difficulty tiers (0-8)
- Distance-based difficulty scaling
- Optional modded content support
- Health/damage multipliers per tier
- Professional implementation

## Configuration
Edit `/data/ms/storage/config.json` to:
- Enable/disable features
- Adjust difficulty curves
- Set spawn rates
- Control modded content

## Installation
1. Place datapack in world's `datapacks` folder
2. Edit config if desired
3. Reload or restart world

Full documentation available in-game via `/trigger ms_help`
